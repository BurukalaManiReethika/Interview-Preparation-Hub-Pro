# ⚡ Developer OS

> All Developer Tools In One Place — no install, no signup, just open and use.

**Live site:** https://interview-preparation-hub-pro.vercel.app/

---

## 🛠 Tools Included

| Tool | Description |
|---|---|
| 🔐 Password Generator | Strong passwords with strength meter & custom options |
| 📱 QR Generator | Turn any URL or text into a downloadable QR code |
| 📄 JSON Formatter | Beautify, minify, and validate JSON with error messages |
| 📝 Notes App | Local notes that persist using `localStorage` |
| 🎨 Color Palette | Random colors, shade variants, HEX/RGB/HSL formats |
| ✍️ Markdown Previewer | Live side-by-side Markdown editor and preview |

## 🚀 Run Locally

Just open `index.html` in your browser — no build step needed!

```bash
git clone https://github.com/BurukalaManiReethika/Interview-Preparation-Hub-Pro.git
cd Interview-Preparation-Hub-Pro
# open index.html in your browser
```

## 🔧 Tech Stack

- Pure **HTML5**, **CSS3**, **Vanilla JavaScript**
- [QRCode.js](https://github.com/davidshimjs/qrcodejs) — QR generation
- [Marked.js](https://marked.js.org/) — Markdown parsing
- [Google Fonts](https://fonts.google.com/) — Inter + JetBrains Mono
- Deployed via **Vercel** + **GitHub Actions**
